# inahurry
定位目前的geolocation，分析出距離最近的火車站，在由node.js去爬台鐵/高鐵的火時刻表，搜尋出最近的發車資訊。

##Install npm
``` text
$ npm install
```
## Stack

- React.js
- Webpack
- Express
- Redux
- Redux-DevTools
- Babel
- postcss

##Start to development server

``` text
# 2 different terminals  
$ npm run webpackDev
$ npm run serverDev
# open this url in your browser
# http://localhost:3000/
# or
# npm run dev
```
##deployment

``` text
# 2 different terminals  
$ npm run webpackDeploy
$ npm run server
# open this url in your browser
# http://localhost:8080/
```
