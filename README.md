# closest-train
定位目前的geolocation，分析出距離最近的火車站，在由node.js去爬台鐵的火時刻表，搜尋出一小時內發車的火車資訊。

##Install npm
``` text
$ npm install
```

##Start to development server

``` text
# 2 ddifferent terminals  
$ npm run weboackDev
$ npm run serverDev
# open this url in your browser
# http://localhost:3000/
```
##deployment

``` text
# 2 ddifferent terminals  
$ npm run webpackDeploy
$ npm run server
# open this url in your browser
# http://localhost:8080/
```
